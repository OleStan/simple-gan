## 1. Статистичні метрики (`metrics.py`)

### 1.1 Порівняння моментів

**Мета:** Порівняння статистик першого порядку (середнє) та другого порядку (дисперсія) між реальним і згенерованим розподілами.

#### Функція: `compute_moment_comparison()`

```python
def compute_moment_comparison(
    real_data: np.ndarray,        # Форма: (n_samples, 2*K)
    generated_data: np.ndarray,   # Форма: (n_samples, 2*K)
    K: int,                       # Кількість шарів
    variance_ratio_threshold: float = 0.3,
) -> MomentComparisonResult
```

**Алгоритм:**

1. **Розділити дані на компоненти σ та μ:**
   ```
   real_sigma = real_data[:, :K]
   real_mu = real_data[:, K:2*K]
   gen_sigma = generated_data[:, :K]
   gen_mu = generated_data[:, K:2*K]
   ```

2. **Обчислити середні значення**
3. **Обчислити дисперсії**
4. **Розрахувати метрики:**
   - Середня абсолютна різниця  
   - Середня відносна різниця  
   - Відношення дисперсій  

5. **Виявлення проблем:**
   - **Колапс мод (mode collapse):** `variance_ratio < (1.0 - threshold)`
   - **Підсилення шуму:** `variance_ratio > (1.0 + 2*threshold)`

**Реальний приклад (Improved WGAN v2, K=51, nz=12):**
```
Mean absolute difference: 0.035113
Mean relative difference: 0.1401 (14.01%)
Variance ratio: 1.0266
Mode collapse: No
Noise amplification: No
Status: ✓ PASSED
```

**Інтерпретація:** Відношення дисперсій ≈1.0 означає, що генератор точно відтворює розкид даних без колапсу мод або нестабільності.

---

### 1.2 Відстань Вассерштейна

**Мета:** Виміряти мінімальну “вартість” перетворення одного розподілу в інший (Earth Mover Distance).

#### Функція: `compute_wasserstein_per_dimension()`

**Формула:**

```
W₁(P, Q) = ∫|F_P⁻¹(u) - F_Q⁻¹(u)| du
```

Менші значення означають ближчі розподіли.

**Реальний приклад:**
```
Mean Wasserstein distance: 0.037662
σ dimensions mean: 0.039854
μ dimensions mean: 0.035471
```

**Інтерпретація:** Низькі значення означають, що згенерований розподіл дуже близький до реального у всіх вимірах.

---

### 1.3 Maximum Mean Discrepancy (MMD)

**Мета:** Ядерна метрика відстані. MMD² = 0 тоді і тільки тоді, коли розподіли ідентичні.

**Формула:**

```
MMD²(P, Q) = E[k(x, x')] + E[k(y, y')] - 2·E[k(x, y)]
k(x, y) = exp(-||x - y||² / (2σ²))
```

**Реальний приклад:**
```
Total MMD: 0.233970
σ component: 0.069567
μ component: 0.164403
```

**Інтерпретація:** Компонента μ вносить більший вклад, що означає складність у відтворенні профілів проникності, але загальне значення MMD низьке.

## 2. Аналіз латентного простору (`latent_analysis.py`)

### 2.1 Обхід латентного простору

**Мета:** Перевірка гладкості та роздільності шляхом варіювання окремих латентних вимірів.

- Середня оцінка гладкості > 0.5

**Реальний приклад:**
```

Inactive ratio: 0.00 (0.0%)
Mean smoothness score: 1.0000
Status: ✓ PASSED
```

**Інтерпретація:** Усі латентні виміри активні та гладкі — генератор ефективно використовує всю латентну розмірність.

---

### 2.2 Стійкість до шуму

**Мета:** Перевірка стабільності генератора при малих збуреннях входу (Ліпшицева неперервність).

**Умова Ліпшицевості:**
```
||G(z₁) - G(z₂)|| ≤ L·||z₁ - z₂||
```

**Критерій проходження:**  
Середня константа Ліпшиця < 10.0

**Реальний приклад:**
```
Mean Lipschitz estimate: 0.8002
Status: ✓ PASSED
```

**Інтерпретація:** Дуже низька константа Ліпшиця (0.8) — генератор надзвичайно стабільний.

## Основні формули

### Порівняння моментів
```
mean_abs_diff = mean(|μ_real - μ_gen|)
variance_ratio = mean(σ²_gen / (σ²_real + ε))
```

### Wasserstein
```
W₁(P, Q) = ∫|F_P⁻¹(u) - F_Q⁻¹(u)| du
```

### MMD
```
MMD²(P, Q) = E[k(x, x')] + E[k(y, y')] - 2·E[k(x, y)]
```

### Ліпшицева константа
```
L = ||G(z + ε) - G(z)|| / ||ε||
```

### Фізичні обмеження
```
σ > 0
μ ≥ 1.0
1e6 ≤ σ ≤ 6e7
1.0 ≤ μ ≤ 100.0
```

## Орієнтири інтерпретації

**Moment Comparison**
- Добре: `variance_ratio ≈ 1.0`
- Колапс мод: `< 0.7`
- Підсилення шуму: `> 1.6`

**Latent Traversal**
- Добре: `inactive_ratio < 0.5`
- Проблема: багато неактивних вимірів → зменшити `nz`

**Noise Robustness**
- Добре: `mean_lipschitz < 10`
